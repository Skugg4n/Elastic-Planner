# Elastic-Planner
Book time, exercise or other works - keep trax of it here!

## Utveckling

1. Installera beroenden: `npm install`
2. Starta dev-server: `npm run dev`
3. Bygg för produktion: `npm run build`

Versionen som visas i UI kommer från `APP_VERSION` i `src/App.jsx` och används även vid export av backupfilen.
